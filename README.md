# Custom_Quadruped
Custom 12-DOF Quadruped Robot Hardware Platform &amp; Control System (ROS 2 Foxy, MPC, MKS_ODrive)
파일 설명  
custom_quadruped/  
├── cad/                # Fusion 360 설계 원본 또는 STEP 파일  
├── config/             # ODrive 설정 코드 및 모터 파라미터 파일  
├── docs/               # 회로도(Schematics), 배선도, 부품 리스트(BOM) (업데이트 예정)  
├── hardware/           # 3D 프린팅 설정값  
├── launch/             # ROS 2 실행을 위한 런치 파일 (업데이트 예정)  
├── meshes/             # 로봇 부품별 STL 파일 (base, hip, thigh, calf 등 분리된 부품은 추후에 업데이트 예정)  
├── scripts/            # 간단한 모터 테스트 스크립트  
├── urdf/               # 로봇 모델 기술 파일 (Xacro/URDF - 업데이트 예정)  
├── LICENSE             # MIT License 파일  
└── README.md           # 프로젝트 개요, 현재 상태, 파일 경로 설명
